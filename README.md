# 🍽️ Sistema de Gestão de Pedidos para Restaurantes

Sistema em tempo real para sincronização entre garçons e cozinha.

## 🚀 Funcionalidades

- **Garçom**: Interface para registrar pedidos por mesa
- **Cozinha**: Tela com pedidos organizados por status (Pendente, Preparando, Pronto)
- **Tempo Real**: Atualização instantânea usando Socket.IO
- **Controle de Tempo**: Monitoramento do tempo de cada pedido

## 📋 Pré-requisitos

- Node.js (versão 14 ou superior)
- NPM ou Yarn

## 🔧 Instalação

1. Clone o repositório
2. Entre na pasta do backend:
   ```bash
   cd backend
   npm install
   ```
